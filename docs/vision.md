# Vision

## Что это

**Singularity** — личный учебный веб-трекер достижения цели. Помогает описать далёкую мечту так, чтобы она не давила масштабом: разбить путь на этапы, зафиксировать честную точку старта и каждый день делать маленькие шаги, которые явно связаны с глобальной целью.

## Для кого

Для одного пользователя — автора проекта. Это не продукт «для всех» на старте, а инструмент для собственной практики и обучения разработке.

## Проблема

Глобальная цель часто выглядит слишком далёкой. Без промежуточной лестницы появляется либо прокрастинация, либо хаотичная «занятость» без ощущения прогресса. Мотивация быстро заканчивается; нужна система.

## Обещание продукта

Пользователь всегда видит:

1. **Куда идёт** — мечта и зачем она нужна.
2. **Где находится** — честная точка А и текущий этап.
3. **Что делать сегодня** — практики и задачи текущего этажа лестницы.
4. **Как понять, что вырос** — критерии перехода на следующий этап.

Связь «сегодняшнее действие → этап → мечта» должна быть видимой в интерфейсе.

## Вдохновение

Продуктовая модель опирается на арку пути Ника из серии Романа Злотникова «Землянин» (особенно первая книга): старт «со свалки», понимание правил системы, учителя и знания, поэтапный рост без перепрыгивания уровней, ежедневная закалка. Это сочетается с проверенными методиками целеполагания и привычек — см. [methodology.md](./methodology.md).

## Сознательно не делаем (на старте)

- Социальную сеть и публичные рейтинги
- Геймификацию ради бейджей
- AI-коуча как обязательную часть
- Мобильное приложение
- Мультипользовательские команды и общие цели
- Маркетплейс курсов / контента

## Критерий успеха MVP

Автор может завести одну реальную мечту, описать точку А, разложить путь на этапы с критериями перехода, вести практики активного этапа и раз в неделю делать короткий обзор — без регистрации и без бэкенда.
