# Мифология сети (продуктовый канон)

Как Singularity **переводит** мотив сети / баз / капсулы из «Землянина» в механику приложения — не копипастом лора, а структурой смысла.

Связано: [methodology.md](./methodology.md), [glossary.md](./glossary.md), [art-direction.md](./art-direction.md).

## Откуда сведения

Структура ниже сверена по открытым фрагментам / тексту первой книги цикла (не дословный пересказ сцен).  
Если найдёшь расхождение — правь раздел «Уточнения читателя».

Мы **не** используем в UI названия «гражданская / военная / инженерная наносеть» и не копируем экономику лутов. Берём *логику системы*.

## Что в книге работает как система (структура)

### 1. Сеть — канал, не скилл

- Почти вся цивилизация крутится через сеть (доступ, учёба, идентичность).
- **Бесплатная сеть** — сильно урезанный доступ: скорее терминал, чем полный канал; базы учатся медленно (почти «как в реале»).
- **Гражданская сеть** — платный/контрактный вход «в нормальную жизнь»; есть **уровни** (первый ставится дёшево / автоматом при работе; выше — от тестов/возможностей).
- Более мощные сети (в тексте цель героя — вроде **инженерной**) — дороже и открывают другой потолок профессий.
- **Военные** ветки мощнее гражданских — нам для продукта не нужны как классы.

Смысл для нас: *качество канала ограничивает скорость и потолок обучения*, но жизнь всё равно идёт через практику и учителя.

### 2. Лакуна и «не ставь сеть слишком рано»

Учитель **откладывает** бесплатную сеть: импланты/канал **усиливают то, что уже есть**, а не делают из нуля героя. Сначала закалка тела/воли, потом канал.

Смысл для нас: онбординг и первые практики важнее «красивого UI сети»; сеть раскрывается по мере движения.

### 3. Базы знаний

- Базу **разучивают** (часто в **медкапсуле**).
- Бытовые базы могут считаться освоенными сразу после разучивания.
- Профессиональные базы требуют **подтверждения** более опытным (принцип «через ступень»: выше рангом экзаменует ниже).
- У баз есть **ранги/уровни**; есть обновления (апгрейд контента базы).
- Бои/работа могут идти **в зачёт практики** освоения — не только капсула.

Смысл для нас: install ≠ certify; drill = наша ежедневная правда.

### 4. Почему «не прогадать с сетью»

Слабый канал = медленная учёба и потолок. Герой копит на более мощную сеть.  
У нас копировать «копить луты и ждать» нельзя — ломает канон **двигаться**.

## Как переводим в Singularity (интеграция, не копипаст)

| Книга (структура) | Singularity (наш термин) | Механика |
| --- | --- | --- |
| Нет сети | **Вне сети** | Онбординг / ещё нет практик |
| Бесплатная урезанная сеть | **Канал доступа** | После первой практики или назначения пути; UI сети уже виден, но «узкий» |
| Гражданская L1+ | **Рабочий канал** | Стабильные нормы, несколько установленных баз |
| Инженерная / мощнее | **Глубокий канал** | Высокий уровень персонажа + подтверждённые рубежи этапа |
| Военная ветка | — | **Не берём** |
| Уровни сети | Полосы **уровня персонажа (XP)** + титул | Без покупки тира |
| Разучивание в капсуле | Экран **Разучивание** (`/drill`) + сессия таймера | Ready/session арты; Moment-диалоги |
| База бытовая | Практика без жёсткого рубежа | Install = первая норма |
| База проф. + экзамен | Практика + **рубеж** / учитель / обзор | Certify отдельно от install |
| Ранг базы 1→2→3 | Грейд практики: minMinutes / «усилена» / серия норм | Апгрейд планки, не новый SKU |
| Подтверждение «через ступень» | Рубеж закрыт *или* учитель этапа / урок недели в обзоре | В соло-приле нет PvP-экзаменатора |
| Луты / цена базы | — | **Не вводим**; цена = минуты и честность |
| Ждать лучшую сеть | Hint: «сначала закалка» | **Нельзя** блокировать движение ради тира |

### Имена канала в UI (наши, не книжные)

1. **Вне сети** — старт  
2. **Канал доступа** — сеть появилась (первая норма / вход в приложение осознанно)  
3. **Рабочий канал** — закалка идёт, базы держатся  
4. **Глубокий канал** — этапные подтверждения + высокий уровень  

Пороги (черновик под N11): XP-уровни и/или «есть подтверждённый рубеж на активном этапе». Точно пропишем при билде.

## Жизненный цикл базы (склейка с ежедневными практиками)

```text
УСТАНОВКА     первая норма/сильно     → узел «установлен»
ЗАКАЛКА       ежедневные check-in     → узел «в закалке / держится»
АПГРЕЙД       ↑ minMinutes / фокус    → узел «усилен»
ПОДТВЕРЖДЕНИЕ рубеж / учитель / обзор → узел «подтверждён»
```

Практика в данных одна (`Practice`). «База» — **проекция состояния** на карте сети.

### Состояния узла (UI)

| Состояние | Условие (черновик) |
| --- | --- |
| Слот | Практика есть, норм ещё не было |
| Установлена | ≥1 `done` / `strong` |
| В закалке | Есть отметки за период |
| Держится | Серия норм или N норм за окно |
| Усилена | `minMinutes` вырос с установки |
| Подтверждена | Связанный рубеж / урок недели закрыт в обзоре |

## Что сознательно отбрасываем

- Валюта, магазин баз, «купил — стал инженером»
- Классы гражданский / военный / инженерный как character class
- Блокировка прогресса, пока не купил «лучшую сеть»
- Отдельная сущность Base рядом с Practice на старте N11
