# Art direction — Moments

Визуальный канон для иллюстраций в choice-диалогах практик (`public/moments/`).  
Нужен, чтобы арты звучали как **одна вселенная пути**, а не «вечная лестница в тумане» и не чужой бренд.

Связано: [methodology.md](./methodology.md), UI-токены в `src/app/globals.css`.

## Принцип

Вдохновение — **дух пути** из «Землянина» (чужие миры, рост компетенцией, учителя, «прокачка из сети»), но:

- **не** копируем персонажей, названия из книги, узнаваемые сцены;
- **не** один-единственный мотив ступеней на все карточки;
- мир Singularity: светлый, стильный, чуть «космический travelogue», без мрачного хоррора.

Лестница компетенции остаётся **смыслом**, а картинка показывает **где** ты сейчас на пути: планета, палуба, капсула сети…

## Свет и тон (важно)

| Делать | Не делать |
| --- | --- |
| Много воздуха, дымка, даль | Почти чёрный кадр «в подземелье» |
| Дневной / золотой час / мягкий корабельный свет | Только ночь и грязь |
| Чистые плоскости металла + живой пейзаж | Захламлённый киберпанк |
| Акцент бирюза / медь / золото овера | Фиолетовый неон, кислотный glow |

Фон UI приложения светлый — арты должны **дружить** с ним: светлее середины кадра, читаемые силуэты.

## Локации-миры (ротация)

Для разнообразия каждый `kind` (и повторные генерации) выбирает **одну** локацию из пула. Не обязательно ступени.

| Локация | Смысл продукта | Как рисовать (стильно, без копий книги) |
| --- | --- | --- |
| **Чужая планета** | Новый этап = новый «мир правил» | Широкий горизонт, странная флора/рельеф, чистое небо или два солнца-намёка; тропа или мост как курс |
| **Орбита / вид с корабля** | Мечта далеко, ты в пути | Иллюминатор, дуга планеты, спокойный свет в кабине; порядок, не аварийный хаос |
| **Палуба / коридор судна** | Ежедневная закалка в «системе» | Длинная перспектива коридора или открытая палуба; линии курса ведут вперёд |
| **Узел сети** | Прокачка из сети / ускорение учёбы | Сдержанные световые нити, узлы, голографическая «дымка знания» — без UI-текста и без Matrix-клише |
| **Медкапсула обучения** | База навыка «заливается», тело в покое, ум растёт | Элегантная капсула / ложе, мягкое бирюзовое свечение панелей, спокойствие; не больница-horror и не аниме-под |
| **Пост наставника** | Голос учителя | Пустая мастерская / смотровая рубка / стол с инструментами роста; человека нет или крошечный силуэт спиной |
| **Ступени / трап** | Не перепрыгивать этап | Один из вариантов, не дефолт на всё: трап корабля, террасы холма, металлические марши в свет |

**Сквозной якорь курса** (вместо «только лестница»): в каждом кадре должен читаться **вектор вперёд** — тропа, линия палубы, луч сети, ось капсулы, перспектива коридора.

## Сеть — как рисовать «прокачку»

Сеть в продукте = ускоритель учёбы (ИИ / знания), не автопилот жизни.

Визуально:

- тонкие **бирюзово-золотые** потоки или соты света;
- ощущение «база легла» — спокойная геометрия, не взрыв скиллов;
- в капсуле: свечение нарастает мягко, лицо/тело не детализируем (анонимно или только контур рук/плеч без лица).

Не рисовать: зелёный код дождём, VR-шлем крупно, рекламный «upload brain» сток.

## Формат файла

- **16:9**, JPEG ~1280px по длинной стороне, ≤250 KB.
- `public/moments/<kind>.jpg`
- Сейчас: `norma.jpg`, `strong.jpg`, `overwork.jpg`, `checkpoint.jpg`

## Настроение по типу момента × локация

Сначала выбери **эмоцию kind**, потом **локацию** из пула (чтобы серия не повторяла один пейзаж).

| Kind | Эмоция | Свет | Идеи кадра |
| --- | --- | --- | --- |
| `norma` | Ступень/сессия закрыта спокойно | Ясный бирюзовый + тёплая медь | Планета: тропа вышла на обзор. Корабль: иллюминатор, курс ровный. Капсула: сессия завершена, свечение ровное |
| `strong` | Запас, овер плана | + золотое свечение «сверх нормы» | Сеть ярче обычного. Палуба в золотом часу. Капсула: более плотный поток базы |
| `overwork` | Слишком долго за целью | Золото резче, воздух всё ещё светлый, но напряжение | Коридор бесконечный. Капсула перегрета (янтарь/золото без крови). Орбита: слишком долгий вахтенный свет |
| `checkpoint` | Ещё на посту | Мягкий дневной / вахтенный | Та же локация, что ушла в сессию; фонарь/панель ещё горит; тишина |

Голос в UI: наставник или **Голос учителя** — не на картинке.

**MOOD examples**

- norma: calm completion, soft teal glow settled, warm copper accent  
- strong: gold overflow energy beyond the plan, still elegant not chaotic  
- overwork: strained gold, too long on duty, still bright not pitch-black  
- checkpoint: quiet watch, soft panels still on, enduring calm  

## Чеклист перед коммитом арта

- [ ] 16:9, без текста и лиц  
- [ ] Не «ещё одна тёмная лестница» без воздуха — есть свет и пространство  
- [ ] Локация из пула; читается вектор курса вперёд  
- [ ] Палитра бьётся с UI; овер = золото, не чёрный блок  
- [ ] Настроение = `kind`  
- [ ] Путь в `MOMENT_IMAGE` / `public/moments/`  
- [ ] Не выглядит как чужой франшизный арт

## Drill / медкапсула (отсек)

Экран `/drill` — hero-арт с персонажем в капсуле (не moment-карточка и не кружок-аватар поверх).

| Файл | Роль |
| --- | --- |
| `public/drill/capsule-{skin}-{male\|female}.jpg` | В капсуле — сессия (live/paused); **та же** комната/капсула, что ready |
| `public/drill/capsule-{skin}-{male\|female}-ready.jpg` | Рядом / готов залезть — idle и после нормы; эталон пары |
| `public/drill/capsule-bay.jpg` | Пустой отсек — fallback без портрета |

Пары генерируются matched: ready = эталон отсека+капсулы; session = тот же сет, только поза персонажа.

Локали капсул: Путник — планета/орбита; Ученик — тёплый пост наставника; Хранитель — **тихий отсек у смотровой палубы** (не проходной коридор); Связной — узел сети.

**Motion + арт:** idle/done → ready-кадр; live/paused → в капсуле + пульс/скан. Таймер не обязан паузиться при уходе с экрана — сессия в данных; на других экранах — `ActiveSessionBar`.

Тот же канон света: не horror-больница, не фиолетовый неон.
