# Backlog

Идеи после MVP. Сюда складываем всё интересное, чтобы не раздувать первую версию.

## После MVP (кандидаты)

- Экспорт / импорт JSON (если не успели в MVP)
- Несколько мечт с явной «фокусной» мечтой
- История изменения Point A (версии честного старта)
- Шаблоны этапов под типичные домены (здоровье, навык, карьера)
- Напоминания в браузере (Notification API)
- Более богатый недельный обзор с графиками
- Теги и поиск по практикам / заметкам
- Режим «не перепрыгивай уровень» — жёсткая блокировка следующего этапа
- Тёмная/светлая тема по желанию (не приоритет)

## Когда-нибудь

- Синхронизация между устройствами / простой бэкенд
- Аккаунт и шифрование данных
- ИИ как «прокачка из сети»: помощник учёбы и декомпозиции этапов (opt-in); всегда через фильтр цели, не автопилот
- Подсказки «это отклонение от курса?» при добавлении практик/задач
- Трекинг «окон обучения» в недельном обзоре
- Мобильное приложение
- Совместные цели / accountability partner
- Более богатая прокачка (квесты-подсказки под мечту) — XP/уровень + карта пути уже есть
- Лёгкий персонаж: имя + титул от прогресса (без RPG-анкеты)
- Геймификация только если усиливает лестницу, не бейджи ради бейджей
- Публичные «карты пути» без личных данных

## Явно не берём без пересмотра vision

- Социальная лента и рейтинги
- Маркетплейс контента
- Замена терапии / медицинские обещания
