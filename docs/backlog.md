# Backlog

Живой список работ. **Не кодим идею, пока она не лежит здесь** (хотя бы одной строкой) и мы не выбрали её в «Next».

## Как работаем

1. **Обсуждаем** идею в чате (зачем, честность, крайние случаи).
2. **Дополняем бэклог** — формулировка + приоритет (`Next` / `Later` / `Won't`).
3. **Реализуем** только выбранный пункт из `Next`.
4. **Переносим в Done** с короткой пометкой, что вышло.

Исключение: мелкий баг/косметика по уже открытой фиче — можно чинить сразу, без нового пункта.

## Done

Сделано после стартового MVP (локально в браузере, без аккаунта).

- Карта пути, XP / уровень / стрик, лёгкий персонаж (имя + титул)
- Онбординг первого захода
- Редактирование этапов, практик, рубежей, источников роста; порядок этапов ↑/↓
- Еженедельные практики (одна отметка на неделю) + сортировка «сделанные вниз» на Сегодня
- Шаблоны этапов: навык / карьера / здоровье
- Напоминания в браузере (пока открыта вкладка)
- Богаче недельный обзор (цифры, дни, weekly-статусы)
- Теги и поиск по практикам
- Строгий режим «не перепрыгивай этапы»
- Несколько мечт + переключение фокуса
- История Point A (снимки при изменении)
- Экспорт / импорт JSON (бэкап персонажа)
- Таймер практик: старт / пауза / продолжить, минимум минут
- Честные уровни усилия: частично / норма / сильно + история 7 дней (heatmap) + тосты
- Авто-settle таймера на границе дня/недели; длинный прогон — спросить при возврате
- Учителя этапа (линия Лакуны): главный наставник, чему учит, урок недели; блок на Сегодня
- Подсказка «эпик ментора → урок недели» (рамка этапа vs узкий кусок)
- Серия норм в HUD (дни подряд с норма/сильно)
