# Backlog

Живой список работ. **Не кодим идею, пока она не лежит здесь** (хотя бы одной строкой) и мы не выбрали её в «Next».

## Как работаем

1. **Обсуждаем** идею в чате (зачем, честность, крайние случаи).
2. **Дополняем бэклог** — формулировка + приоритет (`Next` / `Later` / `Won't`).
3. **Реализуем** только выбранный пункт из `Next`.
4. **Переносим в Done** с короткой пометкой, что вышло.

Исключение: мелкий баг/косметика по уже открытой фиче — можно чинить сразу, без нового пункта.

### N7 — Moment (проработано)

**Проблема:** таймер переваливает норму незаметно; полоса залипает на 100%.

**MVP:**
1. Двухсегментная полоса: норма → овер до 1.5× со свечением/pulse при порогах.
2. Choice-dialog при **норме** и при **сильно** (разные реплики), 1 раз за период.
3. Голос: имя primary teacher, иначе **Голос учителя**.
4. На «сильно»: **плавный** шаг лестницы (25→30) и отдельно **подкалибровка** (~40), каждый — продолжить или закрыть (закрытие сначала по старой норме).
5. Облегчение: −1 ступень (ссылка пока ниже нормы; после «частично»; в диалоге нормы).
6. Иллюстрации в модалке (`public/moments/norma.jpg`, `strong.jpg`).

**Не в MVP:** ИИ-сюжет, длинное дерево веток, звук обязательный.
